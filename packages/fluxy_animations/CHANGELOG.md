## [1.2.1] - 2026-03-07

### Added
- **Initial Release**: High-performance, declarative animations for the Fluxy UI Framework.
- **Elite Animation Primitives**:
    - `FxMeshGradient`: Ultra-modern fluid background with organically moving color blobs.
    - `FxAnimatedBorder`: Premium glowing "light trail" that travels around the perimeter of any container.
    - `FxSpotlight`: Cinematic interactive mask reveal that follows mouse/touch input.
    - `FxGooey`: Metaball-based liquid merge effect where widgets physically "melt" into each other.
    - `FxConfetti`: High-performance organic particle celebrations for interactions.
- **Interactive Physics**:
    - `FxMagnetic`: Widgets that stick to touches and snap back with elite spring physics.
    - `FxPerspective`: 3D perspective tilt effect based on pointer location.
    - `FxJelly`: Elastic distortion (wobble) when widgets are touched.
- **Liquid FX**:
    - `FxLiquidReveal`: Circular transition for revealing content.
    - `FxWave`: Organic liquid fill animation for loading or progress states.
    - `FxLiquidButton`: Metaball-based button that morphs into splitting drops.
    - `FxLiquidShimmer`: Advanced shader-based loading states.
- **Orchestration**:
    - `FxChoreography`: Declarative timeline for chaining multiple animation steps (Scale -> Rotate -> Fade).
    - `FxStaggeredReveal`: Sequential entrance animations for lists.
    - `FxEntrance`: Advanced entry combo (Slide + Scale + Fade).
- **Micro-Interactions**:
    - `FxBoing`: Spring-based button pop.
    - `FxPulsar`: CTA status indicator pulse.
